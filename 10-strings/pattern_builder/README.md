# Pattern Builder

Create a new .NET Core app that can build a pattern as shown below. Let the user choose some options (for example `height` and `width`). Choose the pattern for yourself.

* Rectangle

```text
*******
*******
*******
*******
*******
```

* Pyramid

```text
  *
 ***
*****
```

* Diamond

```text
  *
 ***
*****
 ***
  *
```
